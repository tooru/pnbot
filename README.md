## Bots

* @dpbot https://twitter.com/dpnbot Tweets prime numbers
* @tpnbot https://twitter.com/tnbot Tweets twin prime numbers
* @ipnbot https://twitter.com/ipnbot Reply whether mention is a prime number or not
